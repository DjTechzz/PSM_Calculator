## 日本語

### PSM_Interpolation.ts

この実装は、PSM指標の計算に補間法を使用します。

**主な特徴：**
- 柔軟な入力：事前定義された範囲なしで様々な価格ポイントを扱えます。
- 補間：データのギャップを埋め、より滑らかな曲線を提供する可能性があります。
- 出力：'psm_chart_interpolated.html'を生成します。

**メリット：**
- 異なるデータセットに対してより適応性が高い
- 連続的な価格範囲にわたってより詳細な分析を提供可能
- 不規則な価格間隔のデータセットに対してより正確な可能性がある

**デメリット：**
- 補間によって若干の不正確さが生じる可能性がある
- アルゴリズムがより複雑で、大規模データセットでは遅くなる可能性がある
- 単純で規則的に間隔を置いた価格データには過剰な場合がある

### PSM_Normal.ts

この実装は、PSM分析に固定された価格ポイントのセットを使用します。

**主な特徴：**
- 事前定義された価格ポイント：分析に一定のリストの価格を使用します。
- 直接計算：補間を行わず、与えられたデータを直接使用します。
- 出力：'psm_chart_normal.html'を生成します。

**メリット：**
- より単純な実装で、理解とメンテナンスが容易
- 事前定義された価格ポイントに一致するデータセットの処理が速い
- 特に標準化された調査に対して、より予測可能な結果を提供

**デメリット：**
- 様々な価格範囲に対する柔軟性が低い
- 事前定義された価格の間の重要なデータポイントを見逃す可能性がある
- 事前定義されたポイントに合致しないデータセットでは精度が落ちる可能性がある

### 使用方法

両スクリプトともCSVファイルを入力として受け取ります。以下のように実行してください：

ts-node PSM_Interpolation.ts --csvfile your_data.csv

ts-node PSM_Normal.ts --csvfile your_data.csv
